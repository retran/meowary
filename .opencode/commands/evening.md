description: Evening wrap-up, day summary, and project updates (includes week wrap-up on Fridays)
---

Run the evening routine. This is an interactive process.

Arguments: `/evening`

1. Load `writing` skill and `writing/daily` sub-skill.
2. Follow the evening workflow from the skill:
   - Review tasks: mark done, blocked, dropped, or moved.
   - Collect log entries. Tag with relevant project/person/topic tags.
   - End-of-day scan for stale waiting items (follow-up playbook).
   - Update project/area dashboards. Update weekly note links.
   - KB scan: extract durable facts from today's log, surface resource gaps.
   - Write day summary. Commit.
3. **On Fridays**, the skill auto-includes weekly wrap-up: compile accomplishments, failures, carry-over, reflections, then weekly KB scan.
