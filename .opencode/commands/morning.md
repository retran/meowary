---
description: Morning review, daily note creation, and planning (includes week planning on Mondays)
---

Run the morning routine. This is an interactive process.

Arguments: `/morning`

1. Load `writing` skill and `writing/daily` sub-skill.
2. Follow the morning workflow from the skill:
   - Review yesterday's incomplete tasks, carry over or drop.
   - Create today's daily note from template (if missing). Populate events from `recurring-events.md`.
   - Scan active projects, areas, Jira (if configured), Confluence (if configured).
   - Set today's tasks with drift check. Commit.
3. **On Mondays**, the skill auto-includes weekly planning: create weekly note, review last week, set weekly focus and goals, then continue with the daily flow.
